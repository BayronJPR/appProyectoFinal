Ayuda Registro Instructores







En esta sección del sistema se podrán registrar los instructores;

el sistema solicitará los datos personales así como las disciplinas que entrena,

pronto serán asignado los deportistas.

El botón nuevo limpia los campos para realizar un nuevo registro.

El botón registrar o guardar almacenará la información del instructor,

no pueden haber campos en blanco para el registro
