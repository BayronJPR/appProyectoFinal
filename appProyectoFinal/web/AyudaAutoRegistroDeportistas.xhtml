Ayuda Autoregistro







En esta sección del sistema los usuarios deportistas podrán autoregistrarse;

el sistema solicitará los datos personale asi como las disciplinas que entrenará,

además calculara el Índice de Masa Corporal y grado de obesidad si aplica.

Realizado este autoregistro recibirá un correo electrónico donde se indica que el

proceso de registro se realizo con éxito y que pronto le será asignado un instructor.
